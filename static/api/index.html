Trustpoint API

Machine-readable data endpoints for AI systems and automated tools.

GET /api/project-info.json

Comprehensive project metadata including team, technologies, and features.

Format: JSON

Example: curl https://trustpoint-project.github.io/api/project-info.json

GET /api/resources.csv

Tabular data of all papers, articles, and videos with metadata.

Format: CSV

Example: curl https://trustpoint-project.github.io/api/resources.csv

GET /robots.txt

Crawler instructions and sitemap location.

Format: TXT

GET /sitemap.xml

XML sitemap of all website pages.

Format: XML
